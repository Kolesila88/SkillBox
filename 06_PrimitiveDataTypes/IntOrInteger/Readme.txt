Homework goals
Remember the difference between primitives and objects.
Learn how to get characters (variables of type char) by their code (value of a variable of type int).


Task1
Run the IntOrInteger project code, find the error, and fix the Container class so that the error does not occur.


What is evaluated
Understanding the differences between primitives and objects and correcting the error based on this understanding.


How to submit an assignment for review
Homework needs to be submitted through the Git version control system. Submit a link to the commit with the completed assignment.


Task2
Write in the same project a code that, using a loop and converting numbers into symbols, find all the codes for the letters of the Russian alphabet - uppercase and lowercase,
including the letters Ё.


What is evaluated
Are the codes of all uppercase and lowercase letters of the Russian alphabet, including the letter Ё, found correctly?

Цели домашнего задания
Запомнить разницу между примитивами и объектами.
Научиться получать символы (переменные типа char) по их коду (значению переменной типа int).


Задание 1
Запустите код проекта IntOrInteger, найдите ошибку и исправьте класс Container таким образом, чтобы ошибка не возникала.



Что оценивается
Понимание отличий примитивов от объектов и исправление ошибки с учётом этого понимания.



Как отправить задание на проверку
Домашнее задание необходимо отправить через систему контроля версий Git. Пришлите ссылку на коммит с выполненным заданием.



Задание 2
Напишите в этом же проекте код, который с помощью цикла и преобразования чисел в символы найдите все коды букв русского алфавита — заглавных и строчных, 
в том числе буквы Ё. 



Что оценивается
Верно ли найдены коды всех заглавных и строчных букв русского алфавита, включая букву Ё.
