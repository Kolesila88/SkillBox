What should be done
Run the task in the Main class of the 07_NumbersAndDates / Periods project

Implement the method:

public String getPeriodFromBirthday (LocalDate birthday)

so that when you pass the date of the founding of the Java language, 
it returns the number of years, months, and days from the date of the founding of Java to the present day.
Example, output format:

25 years, 4 months, 6 days

What is evaluated
The correctness of the time period returned by the method and its compliance with the format.
