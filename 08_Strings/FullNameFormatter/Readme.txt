Homework goals
Learn how to work with the indexOf (), lastIndexOf (), substring (), and trim () methods of the String class.


What needs to be done
Execute the task in the Main class of the 08_Strings / FullNameFormatter project

Write a program that accepts the last name, first name and patronymic in one line at the console input (for example, "Ivanov Sergey Petrovich") 
and displays the last name, first name and patronymic separately in the format:

Surname: Ivanov

Name: Sergey

Middle name: Petrovich

A valid string from the user, which we can interpret as a full name, must contain three words, consisting of Cyrillic characters separated by a space, 
may contain a hyphen. 
If the line does not correspond to the format, then the console displays: The entered line is not a full name.

Regular expressions are not allowed in this assignment.

Strictly observe the output format of the result.


What is evaluated
Conformity of the final result to the format specified in the assignment.
Implementing code without using regular expressions.

Цели домашнего задания
Научиться работать с методами indexOf(), lastIndexOf(), substring() и trim() класса String.


Что необходимо сделать
Выполните задание в классе Main проекта 08_Strings/FullNameFormatter

Напишите программу, которая на входе через консоль принимает фамилию, имя и отчество одной строкой (например, «Иванов Сергей Петрович») и выводит фамилию, 
имя и отчество отдельно в формате:

Фамилия: Иванов

Имя: Сергей

Отчество: Петрович

Валидная строка от пользователя, которую мы можем интерпретировать как ФИО, должна содержать три слова, состоящих из кириллических символов, 
разделённых пробелом, может содержать дефис. Если строка не соответствует формату, то в консоль выводится: Введенная строка не является ФИО.

Использование регулярных выражений в данном задании не допускается.

Строго соблюдайте формат вывода результата.


Что оценивается
Соответствие конечного результата формату, указанному в задании.
Реализация кода без использования регулярных выражений.
