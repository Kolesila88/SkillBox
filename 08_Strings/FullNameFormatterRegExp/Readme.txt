
Homework goals:
Learn to write and use simple regular expressions.

What needs to be done:
Execute the task in the Main class 08_Strings / FullNameFormatterRegExp
Write a program (analogue of task 2), which at the input through the console takes the last name, 
first name and patronymic in one line (for example, "Ivanov Sergey Petrovich") and displays the last name, 
first name and patronymic separately in the format:

Surname: Ivanov

Name: Sergey

Middle name: Petrovich


A valid string from the user, which we can interpret as a full name, must contain three words, consisting of Cyrillic characters separated by a space, 
may contain a hyphen. If the line does not correspond to the format, then the console displays: The entered line is not a full name.

Use regular expressions to determine the validity of a string and split it into chunks.

=========================================================================================================================================================

Цели домашнего задания:
Научиться писать и использовать несложные регулярные выражения.



Что необходимо сделать:
Выполните задание в классе Main 08_Strings/FullNameFormatterRegExp
Напишите программу (аналог задания 2), которая на входе через консоль принимает фамилию,
имя и отчество одной строкой (например, «Иванов Сергей Петрович») и выводит фамилию, имя и отчество отдельно в формате:

Фамилия: Иванов

Имя: Сергей

Отчество: Петрович


Валидная строка от пользователя, которую мы можем интерпретировать как ФИО, должна содержать три слова, 
состоящих из кириллических символов, разделённых пробелом, может содержать дефис. 
Если строка не соответствует формату, то в консоль выводится: Введенная строка не является ФИО.

Используйте регулярные выражения для определения валидности строки и её разделения на фрагменты.
