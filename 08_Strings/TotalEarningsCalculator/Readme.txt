Homework goals
Learn how to work with the indexOf (), lastIndexOf (), substring (), and trim () methods of the String class.


What needs to be done
Execute the task in the Main class of the 08_Strings / TotalEarningsCalculator project

Write a code that calculates the amount of earnings of all friends. Use the indexOf (), lastIndexOf (), substring () and trim () methods.
Regular expressions are not allowed in this assignment.


What is evaluated
Correctness of the final result and using only methods for working with substrings in the code without using regular expressions.

===============================================================================================================================================

Цели домашнего задания
Научиться работать с методами indexOf(), lastIndexOf(), substring() и trim() класса String.


Что необходимо сделать
Выполните задание в классе Main проекта 08_Strings/TotalEarningsCalculator

Напишите код, который считает сумму заработка всех друзей. Используйте методы indexOf(), lastIndexOf(), substring() и trim().
Использование регулярных выражений в данном задании не допускается.


Что оценивается
Правильность конечного результата и использование в коде только методов работы с подстроками без использования регулярных выражений.
