
The purpose of the assignment:
Learn to write a configuration for logging events during program execution.

What should be done:
Make three separate logs in the logs folder of the SPBMetro project using log4j2:
	-logs / search.log - fill in information about the stations that are looking for (existing stations),
	-logs / input_errors.log - fill in information about erroneous input (non-existent stations),
	-logs / exceptions.log - log information about exceptions (Exception).

Criteria for evaluation:
"Test" - each file displays only its own type of information.
"Failed" - the task has not been completed.

===============================================================================================================================================================

Цель задания:
Научиться писать конфигурацию для логирования событий при исполнении программы.

Что нужно сделать:
Сделайте три отдельных лога в папке logs проектa SPBMetro с помощью log4j2:
	-logs/search.log — заполнять информацией о станциях, которые ищут (существующие станции),
	-logs/input_errors.log — заполнять информацией об ошибочном вводе (несуществующие станции),
	-logs/exceptions.log — вносить в лог информацию об исключениях (Exception).

Критерии оценки:
«Зачёт» — в каждый файл выводится только свой тип информации.
«Незачёт» — задание не выполнено.
