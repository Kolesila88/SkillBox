
The purpose of the assignment:
Learn to work with the Map collection.

What should be done:
Perform the task in the project
09_ArraysAndCollections / homework_9.4 / PhoneBook

1. Write a program that will act like a phone book:
-If we enter a new name, the program asks for a phone number and remembers it.
-If the new phone number - asks to enter a name and also remembers.
-If we enter an existing name or phone number, the program displays the phone (s) or the name of the subscriber, respectively.
-When you enter the LIST command, the program prints to the console a list of all subscribers in alphabetical order with numbers.

2. Determine name and phone using regular expressions.

3. Think about what to choose as the key and value for the Map, and choose the best option, in your opinion. 
Describe what pluses and minuses you see in your choice.

4. To work with phonebook data, the project contains the PhoneBook class, which should be responsible for storing and working with subscribers. 
Implement all methods and test the class using existing tests. You can add additional methods to the class.

Commands are entered by the user into the console in one line.


Examples of working with the phone book 
Enter a number, name or command:

-Masha

There is no such name in the phone book.
Enter the phone number for the subscriber "Masha":

-79001234567

Contact saved!

Enter a number, name or command:

-79007654321

There is no such number in the phone book.
Enter the subscriber's name for the number “79007654321”:

-Masha

Contact saved!

Enter a number, name or command:

-Nfif @

Invalid input format

Enter a number, name or command:

-LIST

Masha - 79001234567, 79007654321



Criteria for evaluation:
"Pass" - all tests are successfully completed, the phone book works according to the conditions.
"For revision" - the task has not been completed, the tests are completed with an error.

1. Напишите программу, которая будет работать как телефонная книга: 
-Если вводим новое имя, программа просит ввести номер телефона и запоминает его.
-Если новый номер телефона — просит ввести имя и также запоминает.
-Если вводим существующее имя или номер телефона, программа выводит телефон(ы) или имя абонента соответственно.
-При вводе команды LIST программа печатает в консоль список всех абонентов в алфавитном порядке с номерами.

2. Определяйте имя и телефон с помощью регулярных выражений.

3. Подумайте, что выбрать в качестве ключа и значения для Map, и выберите лучший, по вашему мнению, вариант. 
Опишите, какие минусы и плюсы видите в своём выборе.

4. Для работы с данными телефонной книги в проекте находится класс PhoneBook, который должен отвечать за хранение и работу с абонентами. 
Реализуйте все методы и проверьте класс с помощью существующих тестов. Вы можете добавлять дополнительные методы в класс.

Команды вводятся пользователем в консоль одной строкой.
