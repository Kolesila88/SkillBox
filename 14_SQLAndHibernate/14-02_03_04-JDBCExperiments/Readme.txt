
A MySQL server is required to run the application.
On your server, you will need to create and populate a database according to the instructions:

https://www.notion.so/DBeaver-a9619179d54d46fa86a657260d2a58eb

In the hibernate.cfg.xml file, fill in the indicated elements as follows:

        <property name = "connection.url"> Path to your base </property>
        <property name = "connection.username"> Your username in the database </property>
        <property name = "connection.password"> Your password in the database </property>

You can run. =)

        <property name="connection.url">Путь до ващей базы</property>

        <property name="connection.password">Ваш пароль в базе данных</property>
