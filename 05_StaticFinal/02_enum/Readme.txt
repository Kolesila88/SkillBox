
Homework goals
Try to use Enum in practice and remember in which cases it is advisable to use them.
Learn to create and use Enum yourself.

What should be done
Create an ArithmeticCalculator class that will perform arithmetic operations on numbers.
Implement a constructor with two number parameters that stores the passed numbers in the properties of the class.
Implement the calculate method, 
to the input of which one of the arithmetic operations is passed (an object of the Operation class - see the next point of the task),
which needs to be produced with the two numbers passed to the constructor when the object was created.
Create an Enum Operation with three values: ADD, SUBTRACT, MULTIPLY. These values ​​will correspond to arithmetic operations - addition,
subtraction and multiplication.

What is evaluated
All requirements specified in the assignment have been met.
The variable names correspond to the data they store.
The calculate method correctly performs all types of arithmetic operations on the two numbers passed to the constructor when the object was created:
addition, subtraction and multiplication.

Цели домашнего задания
Попробовать на практике использовать Enum и запомнить, в каких случаях их целесообразно применять.
Научиться самостоятельно создавать и использовать Enum.


Что нужно сделать
Создайте класс ArithmeticCalculator, который будет выполнять арифметические операции над числами.
Реализуйте конструктор с двумя параметрами-числами, который сохраняет переданные числа в свойствах класса.
Реализуйте метод calculate, на вход которого передаётся одна из арифметических операций (объект класса Operation — смотрите следующий пункт задания), 
которую нужно произвести с двумя числами, переданными в конструктор при создании объекта.
Создайте Enum Operation с тремя значениями: ADD, SUBTRACT, MULTIPLY. Эти значения будут соответствовать арифметическим операциям — сложению, 
вычитанию и умножению.


Что оценивается
Выполнены все указанные в задании требования.
Имена переменных соответствуют хранимым в них данным.
Метод calculate верно выполняет все типы арифметических операций над двумя числами, переданными в конструктор при создании объекта: 
сложение, вычитание и умножение.
