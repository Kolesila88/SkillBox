
ToDoList web service.

To run the program, you need an installed PostgereSQL database.
In the settings file, specify the path to the database, username and password.
The program runs on local port 8080.

The service implements an api for working with a to-do list, supports CRUD operations.
Frontend - html for demo only.
In the future, a security implementation, a completion mark and a date of relevance will be added.

Web-сервис ToDoList.

Для запуска программы потребуется установленная база данных PostgereSQL.
В файле настроек укажите путь к базе данных, логин и пароль.
Программа работает на локальном порту 8080.

Сервис реализует api для работы со списком дел, поддерживает CRUD операции.
Frontend - html только для демонстрации.
В дальнейшем будет добавлена реализация безопасности, метка завершонносли и дата актуальности.
